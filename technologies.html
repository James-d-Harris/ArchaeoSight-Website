Technologies Used

Dart Dart serves as our primary programming language for the Flutter mobile application, providing excellent performance (due to compiling to native ARM code) and cross-platform capabilities. It also allows for FFI (Foreign Function Interface), which will allow us to write the ML Algorithm in C/C++.
Flutter Our cross-platform mobile framework of choice, using Dart programming language which compiles to native ARM code for high performance on both iOS and Android devices.
TensorFlow Lite A Possible lightweight ML solution for mobile devices that enables real-time predictions in the field, integrated directly into our Flutter application using Dart bindings.
PostgreSQL with PostGIS Our primary database solution with geospatial extensions for storing and analyzing archaeological site data with location information.
Firebase Provides messaging and synchronization services for our offline-first approach, notifying devices when data needs to be updated, with Dart/Flutter integration.
GeoPandas & flutter_map Geospatial libraries with flutter_map providing Dart-based mobile mapping capabilities for archaeological site visualization.
CSV Data Processing Primary format for integrating data from pXRF tools, processed in Dart using CSV parsing libraries for Flutter.
GitHub Our version control and collaboration platform, with GitHub Actions automating our testing pipeline and GitHub Pages hosting project documentation.
VSCode VSCode is a lightweight IDE that offers many plugins for many types of development.
