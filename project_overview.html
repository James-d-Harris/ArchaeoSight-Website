ArchaeoSight: Intelligent Field Support for Archaeological Discovery

Project Overview

ArchaeoSight is a cross-platform mobile application built to assist archaeologists in the field by integrating data from pXRF spectrometers, GPS, and image analysis. It leverages machine learning to predict potential anthropogenic hotspots and enables offline-first functionality to ensure usability in remote locations. Developed in Flutter with TensorFlow Lite, ArchaeoSight aims to modernize how field data is collected, analyzed, and visualized.

Problem & Motivation

Archaeologists today work with high-resolution data including elemental compositions, images, and GPS points. However, these technologies bring logistical issues:

  • Manual recording of data from specialized tools like pXRF devices is tedious and prone to error.
  • Limited or no internet access at dig sites hinders data transfer and collaboration.
  • There’s a need for real-time guidance on where to dig based on collected data.

ArchaeoSight resolves these by streamlining data input, offering offline support, and integrating ML-based hotspot prediction in a user-friendly mobile interface.

High-Level Requirements

Requirements Explanation
Cross-platform mobile support Compatible with both Android and iOS via Flutter for use on any field device.
Machine learning integration Provides real-time predictions of anthropogenic hotspots using TensorFlow Lite.
Offline-first data storage Allows archaeologists to collect and view data without an internet connection, syncing when online.
Integration with pXRF tools Supports CSV data from Bruker and SciAps devices for seamless analysis.

Our Envisioned Solution

ArchaeoSight will offer photo capturing, GPS tracking, and support for uploading CSV exports from pXRF spectrometers. A lightweight ML model will run on-device to provide site predictions based on collected data. The app’s architecture will follow a stream-based offline-first model for robust performance, with synchronization via push notifications when online.

Project Impact

ArchaeoSight provides a major step forward for field archaeology:

  • Identifies excavation hotspots based on real-time ML analysis.
  • Functions fully offline with seamless sync when online.
  • Integrates data from pXRF readings, GPS, and images for comprehensive insight.
  • Works on any modern mobile device, making it ideal for rugged fieldwork conditions.

The system empowers archaeologists to spend less time managing data and more time making discoveries—bridging the gap between AI and anthropology.

The original project was proposed by Dr. Kayeleigh Sharp, who provided use-case data from her excavations at Huaca Letrada in Peru. The vision was refined in collaboration with our team, Jeevana Swaroop Kalapala, and further discussion with Dr. Sharp.
