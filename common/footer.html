Description

This project, titled "Generative Testing of SSD using TLA+ Model-Simulation," aims to enhance SSD validation by leveraging TLA+ model-simulation to generate novel test sequences beyond traditional test plans. By integrating TLA+ with Pluspy and NVMe command execution, the system will identify potential bugs through randomized, reproducible test patterns, optimizing SSD utilization and increasing test coverage. The project, sponsored by Western Digital, seeks to improve reliability, reduce idle hardware time, and provide a robust, well-documented solution for future validation improvements.
