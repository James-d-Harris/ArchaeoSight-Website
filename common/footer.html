Description

ArchaeoSight is a project that aims to aid archaeologists by creating an app to enhance their workflow with specialized tools and data in and off the field, as well as leverage machine learning to predict sites likely to contain sherds and other archaeological fragments. This project is sponsored by Dr. Kayeleigh Sharp at Northern Arizona University.
